# Links

Личная страница-визитка с плашками-ссылками (linktree-стиль).  
Хостится на GitHub Pages: **https://klambiatti.github.io/links/**

---

## Что здесь есть

- Профиль (аватар, имя, описание), иконки соцсетей, карточки-ссылки
- Футер: «since 1986», ротация «Люблю [что-то]», по клику на «since 1986» — счётчик дней с 7 декабря 1986
- Анимации появления, тёмный фон с градиентом и шумом, акцентный цвет

---

## Как править

### Добавить или изменить ссылку-плашку

В `index.html` в блоке `<nav class="links">` скопируй один из `<a class="link-card ...">` и поменяй `href`, текст `link-card__label` и при желании `link-card__desc`. Не забудь добавить класс `anim-item` и свой `style="--delay: N"` (N — следующий по счёту, чтобы анимация шла по порядку).

### Список «Люблю»

В `script.js` массив `ROTATING_TEXTS` — это то, что подставляется после слова «Люблю». Меняй строки, добавляй или убирай.

### Цвет акцента (салатовый, ховеры)

В `style.css` в `:root` переменные `--accent` и `--accent-hover`. Сейчас `#E6FF2B` и `#eeff5c`.

### Аватар, OG-картинка, favicon

- Аватар: замени файл `avatar.png` (рекомендуемый размер 176×176 или 88×88).
- Превью при шаринге: замени `og.png` (рекомендуемый размер 1200×630).
- Иконка во вкладке: замени `favicon.png` (например 32×32 или 64×64).
